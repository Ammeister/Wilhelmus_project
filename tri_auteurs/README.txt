Méthodologie adoptée lors de cette étape


-	Utilisation de VIAF et de DBNL
	o	Ce sont des listes d’autorités
-	On a cherché les données biographiques des auteurs sur ces deux listes (afin de croiser les sources)
	o	Dates de naissances et de morts, périodes potentielles d’activités littéraires
	o	Pour le statut des auteurs (erronés ou non), ces informations sont trop peu souvent mentionnées. Noé propose de compléter le tableur dans la semaine, le temps lui manquait la semaine dernière.
-	La manipulation a été faite manuellement ce qui a pris un certain temps
	o	La mise en place d’un script permettant d’interroger les API de ces bases de données étant trop complexes à faire, le rapporte temps/quantité n’étant pas convenable
-	Une fois les données récoltées et comparées, il a fallu faire un tri en sélectionnant les dates convenables
	o	Selon les sources, ces données pouvaient varier, on a fait le choix de toujours prendre au plus large pour éviter de se limiter
	o	Pour les données manquantes, on a inscrit la notion « NULL »
-	Une fois le tableur épuré, on a dû choisir les auteurs ayant le plus de chances d’avoir écrit l’hymne
	o	Choix purement subjectif, basé sur leur période d’activité et leurs dates
	o	On a fait le choix également de laisser ceux qui étaient peu probable pour n’éliminer personne
-	Enfin, on a exporté ce tableur sous format CSV afin qu’il soit plus aisément traitable par un script
